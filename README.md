Sistema simples de lista de compras em Python com funcionalidades de adicionar, remover e visualizar itens, utilizando estruturas de repetição e listas.
# Lista de Compras em Python

Projeto simples desenvolvido em Python com o objetivo de praticar lógica de programação e manipulação de listas.

## 📌 Funcionalidades
- Adicionar itens à lista
- Remover itens da lista
- Exibir lista de compras
- Menu interativo no terminal

## 🧠 Conceitos utilizados
- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`)
- Listas em Python
- Tratamento de erros (`try/except`)
- Entrada e saída de dados (`input` e `print`)
